# 个人博客开发计划

## 技术选型
- **前端框架**：React + Vite
- **样式方案**：Tailwind CSS + Shadcn UI
- **后端服务**：Express.js
- **数据库**：SQLite（轻量级）
- **部署**：静态网站托管 + 服务器部署

## 核心功能

### 1. 博客管理
- 文章发布、编辑、删除
- 分类和标签管理
- 草稿功能
- 富文本编辑器

### 2. 订阅者管理
- 邮件订阅功能
- 订阅者列表管理
- 邮件模板管理
- 批量邮件发送

### 3. 界面设计
- 响应式布局（PC + 移动端）
- 简约美观的现代设计
- 暗黑模式支持
- 优雅的动画效果

### 4. 大陆用户优化
- 本地化时间显示
- 符合大陆用户习惯的界面元素
- 国内访问速度优化

### 5. 其他功能
- 评论系统
- 阅读统计
- 搜索功能
- RSS 订阅

## 项目结构
```
blog/
├── frontend/           # 前端代码
│   ├── src/
│   │   ├── components/  # 组件
│   │   ├── pages/       # 页面
│   │   ├── hooks/       # 自定义钩子
│   │   └── lib/         # 工具库
│   └── public/          # 静态资源
├── backend/            # 后端代码
│   ├── routes/          # API 路由
│   ├── controllers/     # 控制器
│   ├── models/          # 数据库模型
│   └── utils/           # 工具函数
└── scripts/            # 部署和维护脚本
```

## 开发步骤
1. 初始化项目结构
2. 搭建前端基础框架
3. 实现后端 API
4. 开发核心功能模块
5. 界面美化和优化
6. 测试和部署

## 特色亮点
- 专注于大陆用户体验
- 简约而不简单的设计
- 强大的订阅者管理系统
- 易于维护和扩展
